# OurPets
##宠物特性
*宠物开启后每半分钟动六秒
*拖拽宠物时出现特定动效
*宠物自动贴边
*宠物可设置部分属性
*宠物共提供三种主题
*宠物提供闹钟服务
  *用户可自定义
    *闹钟发生时间
    *闹钟重复情况
    *闹钟提醒消息
  *滑动宠物关闭闹钟
*宠物可接受微信消息（顺便抢红包）
*点击宠物可选择隐藏宠物
  *微信消息		×
  *闹钟提醒		√
  *在Notification中找回宠物
*开机自启
